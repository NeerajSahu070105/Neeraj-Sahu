# Full Stack Web Application

This is a full stack web application developed as part of an assignment.

The project includes a public landing page, an admin panel, and a backend API
to manage projects, clients, contact messages, and newsletter subscribers.

---

## Tech Stack

- Frontend: React (Vite)
- Admin Panel: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB Atlas

---

## Features

### Landing Page
- Hero section
- Projects section with images and dummy action button
- Happy Clients section
- Contact form (Name, Email, Mobile, City, Message)
- Newsletter subscription
- Footer

### Admin Panel
- Add and manage projects
- Add and manage clients
- View contact messages (with mobile and city)
- View newsletter subscribers

### Backend
- REST APIs for Projects, Clients, Contacts, Subscribers
- MongoDB Atlas integration
